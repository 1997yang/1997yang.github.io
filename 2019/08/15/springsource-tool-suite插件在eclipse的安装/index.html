方法一:线下安装

打开help-Install New Software

选择Add

选择Archieve,选择你的spring压缩包。Name的名字随便起。例如：sts3.3.0.Location选择你的压缩包路径

选择OK，弹出如下图，Name中选择4个有Spring IDE。取消Contact all update sites during install to find required software选择（否则安装会很慢，此为联网安装的插件）。

一路Next。安装完毕后出现如下图。点击Yes后，eclipse会重启。

在window里选择Preferences

出现Spring和SpringSource,说明安装成功.
